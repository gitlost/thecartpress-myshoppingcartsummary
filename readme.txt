== Description ==

TheCartPress My Shopping Cart Summary is a plugin for TheCartPress. It adds an alternative, extended version of the native TCP Shopping Cart Summary widget.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload My TCP Shopping Cart Summary to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in the WordPress admin dashboard.
3. Go to the 'Appearance' > 'Widgets' menu in the WordPress admin dashboard and drag the 'My TCP Shopping Cart Summary' widget to a widget area and configure and save.
